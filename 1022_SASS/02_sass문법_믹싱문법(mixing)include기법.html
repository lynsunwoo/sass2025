02_sass문법_믹싱문법(mixing)include기법

1. $mixin기법

웹 페이지 전체에 재사용할 css코드를 정의하는 문

    @mixin mixin명{
      //반복해서 사용할 스타일 서식을 적용한다.
    }

2.include문법

@mixin을 선택자에 포한하는데 사용하는 지시문

    선택자{
      @inculde mixin명;
    }

3. @mixin을 중첩하여 새로운 @mixin만들기

    @mixin 믹스인명 A{
       //재사용할 스타일 서식
    }
    @mixin 믹스인명 B{
       //재사용할 스타일 서식
    }
    @mixin 믹스인명 C{
      @include 믹스인명 A;
      @include 믹스인명 B;
    }
